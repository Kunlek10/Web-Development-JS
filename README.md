# JS Utilities

A simple modern web app with three JavaScript tools.

## Tools

- **Reverse String** — type any string and get it reversed
- **Palindrome Check** — enter a number to check if it reads the same forwards and backwards
- **Restaurant Bill** — enter a total cost and tip percentage to calculate the final bill

## How to Run

Open `index.html` in your browser — no setup or dependencies needed.

## Files

- `index.html` — page structure
- `style.css` — styling and animated background
- `index.js` — JavaScript logic for all three tools
